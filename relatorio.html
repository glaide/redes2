Relatório do trabalho

Conexão com o servidor

    Este trabalho consiste em implementar um protocolo de comunicação entre dois dispositivos, um servidor e um cliente, utilizando um endereço IPv4 e uma
    interface de STREAM (TCP/IP), que possui uma conexão confiavel, e permite a comunicação entre dois sockets (que é o requisito deste trabalho).
    Na implementação desse trabalho é feita com o uso de dicionarios em python, pois o mesmo utiliza hash para busca de dados na tabela.

    Código fonte:
    Cliente Servidor cache Servidor Patagonia Servidor Saara Servidor Antartida

Servidores de temperatura

    Nesta comunicação há 3 servidores de temperatura: Antartida, Patagonia e Saara.
    Cada servidor de temperatura envia o valor atual da temperatura para quem está solicitando, e logo após gera-se um novo valor para a temperatura local.

    As tarefas dos servidores de temperatura são: aguardar uma solicitação, aceitar a conexão com quem está fazendo
    a requisição, responder com a temperatura do local e gerar uma nova temperatura.

Funcionamento do servidor

    O servidor deve ser iniciado com o comando:
    python3 Servidor.py

    O servidor deve ser iniciado em um terminal separado.
    O endereço IP do servidor é o 'localhost' e a PORTA é a porta 5000.

Configurações dos servidores de temperatura

    Os endereços IP dos servidores são:

    HOST_saara = '127.0.0.1'

    HOST_antartida = '127.0.0.1'

    HOST_patagonia = '127.0.0.1'

    As portas dos servidores são:

    PORT_saara = 5001

    PORT_antartida = 5002

    PORT_patagonia = 5003

Funcionamento principal do servidor

    O servidor permanece aguardando a requisição do cliente, e ao receber a requisição, ele aceita a conexão.
    Após aceitar a conexão e receber a mensagem, o servidor faz uma verificação para saber se o tipo da mensgem está certo,
    caso esteja, ele atualiza a tabela cache, monta a mensagem a ser enviada e faz seu envio.
    Caso o tipo da mensagem seja errada, o servidor envia a seguinte mensagem 'tente outro comando'.
